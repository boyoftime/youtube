Get Started with Inviloader

Inviloader is a revolutionary tool allowing you to download YouTube videos directly to your PC using a simple CMD command. Save videos for offline viewing, backup content, or repurpose for personal use - fast and efficiently.

The program works by integrating with FFmpeg, a powerful multimedia framework, to handle video and audio processing. Simply enter a YouTube video link into the command line, and Inviloader does the rest.

  • Supports multiple video resolutions and formats.
  • Simple, user-friendly command-line interface.
  • Free version with robust features.

How It Works!

User-friendly menu: Choose '1' for Video or '2' for Audio.

Select video quality up to 1080p HD. Inviloader handles the download automatically.

Files are saved automatically in your 'Downloads/Inviloader output' folder.

Downloading audio is simple: Choose the audio option and paste the YouTube link.

Installation Guide: FFmpeg

Inviloader requires FFmpeg to process video and audio. Follow these steps to install it.

For Windows

Step 1: Download & Extract FFmpeg

Download FFmpeg for Windows
  1. Extract the downloaded file (e.g., using 7-Zip) to a permanent location, like C:\ffmpeg.
  2. Find the bin folder inside (e.g., C:\ffmpeg\bin).
  3. Add FFmpeg to your system's PATH:
    • Search for "Environment Variables" in the Start menu and open "Edit the system environment variables".
    • Click "Environment Variables...".
    • Under "System variables", select "Path" and click "Edit...".
    • Click "New" and paste the full path to the bin folder (e.g., C:\ffmpeg\bin).
    • Click OK on all windows to save.
  4. Verify Installation: Open a new Command Prompt or PowerShell window and type ffmpeg -version. You should see FFmpeg details.

For macOS

Option 1: Using Homebrew (Recommended)

  1. Install Homebrew if you don't have it. Open Terminal and run:
    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
  2. Install FFmpeg: brew install ffmpeg

Option 2: Manual Download

Download FFmpeg for macOS
  1. Extract the downloaded file to a location like ~/ffmpeg.
  2. Add to PATH (adjust based on your shell, usually zsh):
    echo 'export PATH="$HOME/ffmpeg/bin:$PATH"' >> ~/.zshrc && source ~/.zshrc

Verify Installation: Open a new Terminal window and type ffmpeg -version.

For Linux

Option 1: Using Package Manager (Recommended)

  1. Ubuntu/Debian: sudo apt update && sudo apt install ffmpeg
  2. Fedora: sudo dnf install ffmpeg
  3. Arch Linux: sudo pacman -S ffmpeg
