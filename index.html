FFmpeg Required

Inviloader requires FFmpeg to function properly. Click the button below to download FFmpeg:

Download FFmpeg

Follow the instructions below to install FFmpeg on your computer:

  1. Download FFmpeg using the button above.
  2. Extract the contents of the zip file to a folder on your computer.
  3. Navigate to the folder where you extracted FFmpeg.
  4. Open the Command Prompt or Terminal and add the FFmpeg bin folder to your system's PATH environment variable.
  5. Test the installation by typing ffmpeg -version in your command line.
